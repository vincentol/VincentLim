Relevant Completed Courses


Intro to CS & OOP: Java (Accelerated Pace)

In this class, I learned the basics of Object Oriented Programming through Java. We did 10 projects throughout the quarter varying from animating 'critters' to drawing triangles on the terminal. Everything was done in a UNIX environment using VIM.


Basic Data Structures and Object-Oriented Design

The most important things in Object-Oriented Design are Data Structures, and CSE12 taught me the importance of data structures by having me design some basic ones. I now use these data structures in my everyday programming.


Computer Organization and Systems Programming

A good programmer can write code, but a great programmer understands how that code is being run by the machine. CSE 30 taught me how code is translated to Assembly language. CSE 30 made me more appreciative of higher level languages such as Java, C, and C++ by having me write my programs in ARM Assembly.


Advanced Data Structures

Simple data structures are useful, but can create complicated code because they are inefficient. That is why we must learn about advanced data structures. By building more complex data structures like trees and tries, I learned how to design data structures based on what my program needs.


Software Engineering

CSE 110 taught me the agile process, one of the most common methods that companies use to make software. I was able to put this process to use in our quarter long class project (see projects section for more information).


Computer Architecture

These two classes taught me how computers run our code: How code gets translated multiple times until it is binary, then the machine reads it using different circuits. Some might think that programmers don't need to know how the machine runs the code, they just have to make the code. But by learning how the machine works, I now understand that my programs can be optimized for speed by how you write them, what ISA you are using, and what Hardware(processor) model the machine is running.

Class Projects

Budgetting App
For CSE110 (Software Engineering), I, along with a group of 4, built a functional, very basic budgetting app for Android Devices using FireBase and Android Studio. This app is not available on the webstore because of how primitive it is, however, the files for this application are available to view on github.
Features for this app include: tracking spending, viewing spending history by categories, labeling spending activities, and setting reoccuring bill payments.

9-bit Processor
For CSE141L (Introduction to Computer Architecture Lab), I joined a group of 3 members in order to build a fully functional 9-bit processor model using System Verilog. This processor was optimized for three programs: Product of A*B*C, StringMatch, and ClosestPair. The code for this 9-bit processor is not available for the public because of UCSD's Academic Integrity Policies, but the code is available upon request.
